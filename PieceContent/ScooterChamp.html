👩‍💻 Team Size 2
⚙ Tools Unity
Visual Studio
Git
Adobe Photoshop

Overview

This project was a submission for the Indie City Games Allstars 2025 game jam. The following was created with the theme of speed, with the main requirement being a timer and leaderboard. For this project I designed, collaborated, programmed, and delegated tasks to ensure the game contained all necessary features and functionality.

Goal

Since this was my first participation in a game jam, my main goal was to create a game that fit the theme but also bring a childhood equipment from gym class: floor scooters. The primary object is to get the fastest time, and performing tricks allows the player to gain a speed boost. I wanted the style to be similar to Subway Surfers, where players navigate through obstacles and only have three positions to move between.

Task Responsibilities

  • Designed and implemented four UI menus to provide a user-friendly experience for players. This included creating a main menu, pause menu, quit menu, and leaderboard menu. Each menu was designed to be visually appealing and easy to navigate, with clear instructions and feedback for the player. Along with the menus, I also handled scene transitions using Dotween to create smooth fades animations.
  • Developed a hallway generator script to determine the level and size. This allows us to quickly test in the inspector how many hallways we want to generate.
  • Implemented a local leaderboard system. This allowed the user to enter their name with the size of 3 characters and produce a randomize ID. If the player had a faster time within the top ten, they would be placed accordingly. The leaderboard data is also saved using JSON to ensure the data doesn’t erase when the player quits.
  • Assisted with development of the timer: such as stopping when the player reached the end, if the pause menu is displayed, and if the player can be placed on the leaderboard.
  • Created all the UI visuals, such as background images and tricks reactions using Adobe Photoshop.
  • Delegating tasks to team members to ensure efficient workflow, tracking progress, and maintaining clear communication throughout the project. This included assigning specific tasks, setting deadlines, and providing support and guidance as needed.
  • Researching and collecting assets for the game to fit the overall theme and style.
